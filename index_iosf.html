Hoy hemos trabajado en la práctica grupal de desarrollo web siguiendo la metodología Agile. Cada integrante ha creado su propia rama feature_NombreApellido a partir de la rama develop del repositorio del grupo. Hemos diseñado y estructurado nuestra web con:
Un header con logo, lema y navegación a secciones internas. Un main con tres secciones: información, reservas y contacto. Un footer con nuestro nombre y enlace a GitHub.
Durante el desarrollo, se realizaron commits frecuentes para guardar los avances y, al finalizar, cada miembro creó un Pull Request hacia la rama develop del repositorio original.
Esta práctica nos permitió practicar la colaboración en equipo, el control de versiones con Git y GitHub, así como aplicar nuestros conocimientos de HTML y CSS en un proyecto real.
